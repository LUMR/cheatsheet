# Linux 命令

| 命令 | 说明 |
|------|------|
| `ls -la` | 列出文件（含隐藏） |
| `cd <path>` | 切换目录 |
| `pwd` | 当前目录路径 |
| `mkdir -p <dir>` | 递归创建目录 |
| `rm -rf <dir>` | 强制删除目录 |
| `cp -r <src> <dst>` | 递归复制 |
| `mv <src> <dst>` | 移动/重命名 |
| `cat <file>` | 查看文件内容 |
| `tail -f <file>` | 实时追踪文件 |
| `grep -r <pattern> <dir>` | 递归搜索内容 |
| `find <dir> -name <pattern>` | 查找文件 |
| `chmod +x <file>` | 添加执行权限 |
| `curl -O <url>` | 下载文件 |
| `wget <url>` | 下载文件 |
