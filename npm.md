# npm / pnpm

| 命令 | 说明 |
|------|------|
| `npm init` | 初始化项目 |
| `npm install` | 安装依赖 |
| `npm install <pkg>` | 安装指定包 |
| `npm install -D <pkg>` | 安装为开发依赖 |
| `npm run <script>` | 运行脚本 |
| `npm update` | 更新依赖 |
| `npm cache clean --force` | 清除缓存 |
| `pnpm install` | 安装依赖（更快） |
| `pnpm add <pkg>` | 安装指定包 |
| `pnpm add -D <pkg>` | 安装为开发依赖 |
