# 快捷键

## VS Code

| 快捷键 | 说明 |
|--------|------|
| `Ctrl + P` | 快速打开文件 |
| `Ctrl + Shift + P` | 命令面板 |
| `Ctrl + `` ` `` | 打开终端 |
| `Ctrl + /` | 注释/取消注释 |
| `Ctrl + D` | 选中下一个相同词 |
| `Alt + ↑/↓` | 上/下移动行 |
| `Ctrl + Shift + K` | 删除行 |
| `Ctrl + B` | 切换侧边栏 |
| `Ctrl + \` | 拆分编辑器 |

## Windows

| 快捷键 | 说明 |
|--------|------|
| `Win + D` | 显示桌面 |
| `Win + E` | 打开文件管理器 |
| `Win + L` | 锁屏 |
| `Win + Tab` | 任务视图 |
| `Alt + Tab` | 切换窗口 |
| `Ctrl + Shift + Esc` | 任务管理器 |
